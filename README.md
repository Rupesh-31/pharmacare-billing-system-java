# PharmaCare Billing System (Java)

## Overview
PharmaCare Billing System is a Java-based console application that simulates a pharmacy billing workflow. It captures patient and medicine details, calculates total amounts, and generates date-stamped invoice files using core Java features.

## Features
- Patient and medicine data management
- Automatic bill and invoice generation
- Total amount calculation
- Payment mode handling (Cash / UPI / Card)
- File-based invoice storage

## Technologies Used
- Java (Core Java)
- OOP Principles
- Java Collections
- File Handling
- Date & Time API

## Project Structure
medicineBilling/
├── Patient.java
├── Medicine.java
├── Bill.java
└── MedicineBillGenerator.java

## How to Run
1. Clone the repository
2. Run `MedicineBillGenerator.java`
3. Enter details via console
4. Invoice file is generated at the given location
